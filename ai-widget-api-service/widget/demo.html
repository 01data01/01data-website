üîß Setup Instructions

To enable voice chat, you need to:

  1. Create an ElevenLabs account and set up a Conversational AI agent
  2. Get your agent ID from the ElevenLabs dashboard
  3. Set the ELEVENLABS_AGENT_ID environment variable in your deployment
  4. Or pass the agent ID directly in the widget configuration
